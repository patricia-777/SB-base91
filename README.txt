Como rodar o programa (forma 1):
	1) No terminal digite --> gcc base91.c
	2) Execute o programa
	3) Digite a opção de funcionalidade que você quer:
		--> c para codificar para a base91
		--> d para decodificar da base91 para o original
	4) Escreva o nome do arquivo que você quer que seja a entrada
	5) Escreva o nome do arquivo que você deseja que seja a saída

	OBS.: Escreva as respectivas extensões dos arquivos, não se esqueça!

Como rodar o programa (forma 2):
	1) No terminal digite --> gcc base91.c
	2) Execute o programa passando por linha de comando os argumentos da seguinte forma:
		--> programa.out  opção  arquivo_de_entrada.ext  arquivo_de_saida.ext
